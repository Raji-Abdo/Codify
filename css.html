CSS

Le CSS est un langage informatique utilisé sur l'internet pour mettre en forme les fichiers HTML ou XML. Ainsi, les feuilles de style, aussi appelé les fichiers CSS, comprennent du code qui permet de gérer le design d'une page en HTML.

Les Bases De Langage CSS

Il existe trois manières d'insérer une feuille de style :
- CSS externe (fichiers .css)
- CSS interne (<style>)
- CSS en ligne (style=" ").




Voici quelques-unes des Propriétés CSS essentielles :


Les Couleurs

Les couleurs sont spécifiées à l'aide de noms de couleurs prédéfinis ou de valeurs RVB, HEX, HSL, RGBA, HSLA.


Vous pouvez définir la couleur d'arrière-plan des éléments HTML :
  <h1 style="background-color:Blue;">Texte Bleue</h1>
  <p style="background-color:Green;">Texte verte</p>

Les Marges

- Les propriétés de marge CSS sont utilisées pour créer
   de l'espace autour des éléments, en dehors de toute
   bordure définie.
- Avec CSS, vous avez un contrôle total sur les marges. Il existe
   des propriétés permettant de définir la marge
   de chaque côté d'un élément (haut, droite, bas et gauche).

Les Bordures

- Les propriétés de bordure CSS vous permettent de spécifier
   le style, la largeur et la couleur de la bordure d'un élément.
- Vous pouvez changer la couleur et le style des côtés de la
   bordure individuellement
